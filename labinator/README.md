# Labinator has moved

Labinator is now **Poiesis** — part of the [Praxis Suite](https://github.com/PraxisSuite).

## New Home

> **[github.com/PraxisSuite/Poiesis](https://github.com/PraxisSuite/Poiesis)**

Poiesis is the evolution of Labinator — a command-line provisioning wizard for Proxmox VE homelabs. Same tool, better name, new home.

## Why the Move?

Labinator grew from a small personal script into a full-featured provisioning system. Along the way it found two companion tools — one for moving and packaging configs, one for documenting infrastructure. Together they form the **Praxis suite**: an ecosystem of homelab management tools built around the idea that a homelab should not be a second job.

The name Poiesis comes from Aristotle's concept of *poiesis* — the act of making, bringing something into being. It fits.

## The Full Suite

| Tool | What It Does |
|---|---|
| [Poiesis](https://github.com/PraxisSuite/Poiesis) | Provisions LXC containers and QEMU VMs on Proxmox |
| [Poreia](https://github.com/PraxisSuite/Poreia) | SSH/SFTP file manager and deployment package builder |
| [Theoria](https://github.com/PraxisSuite/Theoria) | Automatically documents your entire homelab |

All development continues at [github.com/PraxisSuite](https://github.com/PraxisSuite).
