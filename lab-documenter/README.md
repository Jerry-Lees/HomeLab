# Lab Documenter has moved

Lab Documenter is now **Theoria** — part of the [Praxis Suite](https://github.com/PraxisSuite).

## New Home

> **[github.com/PraxisSuite/Theoria](https://github.com/PraxisSuite/Theoria)**

Theoria is the evolution of Lab Documenter — a comprehensive infrastructure documentation system that automatically discovers and documents your entire homelab. Same tool, better name, new home.

## Why the Move?

Lab Documenter grew from a documentation script into a full-featured infrastructure discovery and documentation engine. Along the way it found two companion tools — one for provisioning infrastructure, one for moving and packaging configs. Together they form the **Praxis suite**: an ecosystem of homelab management tools built around the idea that a homelab should not be a second job.

The name Theoria comes from Aristotle's concept of *theoria* — observation, contemplation, seeing clearly. It fits.
